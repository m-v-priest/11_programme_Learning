成功连接到 MySQL 后，你可以在命令行中直接执行 SQL 查询。

列出所有可用的数据库： +
SHOW DATABASES;

选择要使用的数据库： +
USE your_database;

列出所选数据库中的所有表： +
SHOW TABLES;

退出 mysql> 命令提示窗口可以使用 exit 命令，如下所示： +
EXIT;

或者使用： +
QUIT;
