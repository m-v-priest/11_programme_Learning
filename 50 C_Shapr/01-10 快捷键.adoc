== 行操作

[options="autowidth"]
|===
|Header 1 |Header 2

|复制本行到下一行上
|Ctrl + D

|移动行
|alt + 上下键

|将选中行, 往下移一行位置
|Alt+Shift+T

|删除当前行
|Shift+Delete
|===

== 匹配括号

|===
