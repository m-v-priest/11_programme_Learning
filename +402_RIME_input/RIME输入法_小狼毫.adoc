编辑 installation.yaml 文件

....
sync_dir: C:\phpStorm_proj\11_programme_Learning\+402_RIME_input\rimeSync

上面的路径,修改为你的云盘路径. (注意, 路径中不能用中文!)


image:/img/022.png[,%]


保存文件并重新部署（右键托盘图标 -> “重新部署”）。

在主设备上，右键托盘图标，选择“用户数据同步”。这会将当前设备的配置和用户词典打包到一个压缩文件存放在 sync_dir 指定的目录中。

image:/img/024.png[,%]

在另一个设备上，重复上面步骤，确保 sync_dir 指向同一个云盘目录。

在第二个设备上，同样执行“用户数据同步”。Rime 会自动合并云端和本地的数据。
