== 新建链接页面 -> [[]]

方法1 : 直接把连接页面的文件名, 放在[[ ]]中. +
方法2 : 先输完文件名, 选中它, 再按[ 键2次.

每个[[ ]]中的文字都是一个链接，也都是一个新的笔记页面。
